Conoce el Elemento "small" de HTML

En el mundo del desarrollo web, la presentación de contenido es una parte esencial para captar la atención de los usuarios y comunicar de manera efectiva. Una de las herramientas que HTML nos proporciona es el elemento "small", que nos permite resaltar información de manera discreta y sutil. En este artículo, exploraremos cómo usar este elemento para realzar ciertos fragmentos de texto mientras mantenemos la armonía visual de nuestra página.

¿Qué es la etiqueta "small"?

El elemento "small" en HTML se utiliza para indicar que el texto contenido en su interior es de tamaño reducido en comparación con el texto circundante. Aunque se utiliza principalmente para reducir el tamaño de la fuente, su propósito no es simplemente hacer que el texto sea más pequeño, sino más bien resaltar información secundaria, legal o notas a pie de página de una manera que no distraiga al lector.

Usos Comunes

Notas de Pie de Página

Uno de los usos más comunes del elemento "small" es para notas a pie de página o aclaraciones. En lugar de interrumpir el flujo del contenido con información adicional, puedes utilizar "small" para proporcionar detalles adicionales de manera discreta:

Observa

<p>
        El descubrimiento se realizó el 15 de junio de 2023
        <small>(ver Informe Científico XYZ)</small>
        </p>

Informacion Legal

En muchas páginas web, encontrarás información legal como derechos de autor, licencias o exenciones de responsabilidad. En lugar de destacar en negrita o en tamaños de fuente más grandes, puedes utilizar "small" para mantener estos detalles presentes pero no intrusivos:

Observa

<footer>
        <p>odos los derechos reservados © 2023 <small>(Nombre de tu Empresa)  </small>  </p>
        </footer>

Atribiciones y Creditos

Cuando estés presentando contenido que incluya atribuciones a otros autores, fuentes o recursos, "small" puede ayudarte a mantener el flujo visual de la página mientras das crédito adecuado:

Observa

<blockuote>
        <p>"La creatividad es contagiosa, pásala." - Albert Einstein </p>
        <p><small>Fuente: Discurso en la Universidad de Princeton, 1955  </small></p>
        </blockuote>

Algo curioso de la etiqueta "small"

Imagina a un diseñador web rockero que tenía que contar las historias de un antiguo edificio en un sitio web. Pero había detalles pequeños y juguetones que no quería que se perdieran en la multitud. ¡Bingo! El diseñador usó la etiqueta <small>. Así que, en lugar de solo decir "Construido en 1892", lo hizo "Construido en <small>1892</small>". ¡Fue como un secreto cómplice para los ojos curiosos!

Principales atributos de la etiqueta "small"

Esta tabla resume los atributos principales que puedes usar con el elemento small. Cada atributo te permite personalizar, identificar o estilizar el contenido dentro del elemento small de acuerdo a tus necesidades.

Atributo Descripción Ejemplo
class Clase para estilos <small class="destacado" > Text </small>
id Identificador único <small id="nota" > Nota </small>
style Estilo en línea <small style="font-size":80% >Texto </small>

En resumen:

  • Tamaño reducido: Muestra texto en un tamaño de fuente más pequeño
  • Información secundaria: Utilizada para notas o detalles menos importantes.
  • Contraste visual: Crea diferenciación en el diseño.
  • Legibilidad:Aunque pequeño, el texto sigue siendo legible.
  • Jerarquía: Indica contenido de menor importancia.
  • Accesibilidad : Ayuda a usuarios de lectores de pantalla.
